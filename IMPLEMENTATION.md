## 技术实现

### 使用的技术栈
- **框架**：Lynx (React-like框架)
- **组件**：view, text, image, scroll-view
- **样式**：CSS（Flexbox布局）

### 主要文件
- `src/App.tsx` - 主应用入口，渲染餐厅卡片
- `src/RestaurantCard.tsx` - 餐厅卡片组件
- `src/RestaurantCard.css` - 卡片样式文件
- `src/assets/` - 图片资源文件夹

### 关键特性
1. **响应式布局**：使用Flexbox实现灵活布局
2. **横向滚动**：使用 `scroll-view` 组件实现菜品横向滚动
3. **图片资源管理**：所有图片使用本地资源导入
4. **样式隔离**：每个组件有独立的CSS文件

## 运行项目

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
